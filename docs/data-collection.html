We have collected a total of 5566 cells from a total of 54 individuals.

Column descriptions:

  • Individual - This includes any individual that had at least 5 cells that passed the quality control cutoffs

  • Number of cells - Total number of cells for that individual that passed the quality control cutoffs

  • Median number of genes detected - This only includes human genes and ERCC spike-ins which passed the gene filtering cutoffs

  • Median number of molecules - This only includes the molecules that map to human genes and ERCC spike-ins which passed the gene filtering cutoffs
