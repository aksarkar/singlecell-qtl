2017-10-25

Last updated: 2018-02-22

Code version: 2df7b00

We have collected a total of 5012 cells from a total of 54 individuals.

Column descriptions:

  • Individual - This includes any individual that had more than 5 cells on at least 1 C1 chip.

  • Number of cells - Total number of cells for that individual. This is a high estimate because only a few QC filters were used. The number will decrease when more stringent QC filters are applied.

  • Median number of genes detected - This only includes human genes which were observed in at least 50% of the cells across the entire experiment.

  • Median number of molecules - This only includes the molecules that map to human genes which were observed in at least 50% of the cells across the entire experiment.
